# Menu

Dropdown menu attached to a trigger. Items inherit the `DropdownItem` styling
(see Select): `rounded-xs` cells, hover / focused / selected all use
`bg-brand-50 text-brand-700`.

## Row actions

<Canvas of={MenuStories.RowActions} />

The canonical pattern: `…` / `Actions` button next to a row, producing
Duplicate / Archive / Delete. `intent="danger"` flags destructive items.

## Sections

<Canvas of={MenuStories.Sections} />

Group related items under a `MenuSection title="…"` with a `MenuSeparator`
between. Useful for filter toggles and multi-category actions.
