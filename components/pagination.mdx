# Pagination

Pagination is **navigation chrome**, not action. It lives under long listings
(devices, plans, audit log, customer requests) and lets the user move through
pages without leaving the table's context.

It intentionally does **not** reuse `Button`. Page numbers are not buttons in
the product-decision sense — they're a row of locations. Using the heavy
button styling pulled the user's eye away from the table rows and made the
current page hard to spot among a crowd of outlines. Pagination ships its own
compact, ink-only styling instead.

## At the start / end

When the user can't move further back or forward, the corresponding
First / Previous or Next / Last controls drop their hover affordance and
dim to `ink-300`. They stay visible so the row never reflows, but they don't
invite a click.

## Few pages

Under 8 pages, every page renders without a gap. No ellipsis, no truncation.

## Rules

- **Height.** Every target is 32px tall (admin touch target minimum).
- **Radius.** `rounded-xs` (4px) on every target and the current-page chip.
- **Type.** `font-body`, 600, 13px, `tabular-nums` on the digits so the row
  doesn't reflow between pages 9 and 10.
- **Current page.** Filled `bg-brand-500` + white — the single loud chip in
  the row. Using the primary brand color here (rather than neutral ink) makes
  the user's location unmistakable against a busy table.
- **Other pages.** `text-ink-700`, no border. Hover fills `bg-brand-50` and
  shifts to `text-brand-700` — high enough contrast to feel tactile without
  competing with the active chip.
- **Nav controls.** 32px squares with a 1px `ink-200` border. Hover picks up
  the brand treatment: `border-brand-500`, `bg-brand-50`, `text-brand-700`.
  Disabled (no `href`) loses hover and dims to `ink-300`.
- **Ellipsis.** `ink-500`, non-interactive, same 32px footprint so spacing
  stays on the grid.
- **Focus.** 3px `brand-200` outline with a 2px offset — matches the rest of
  the system.
- **No emphasis animation.** No hover lift, no colour pulse. Pagination is
  plumbing; don't draw attention to it.
