# Cards

The core surface of Cellestial — every plan, device, and bundle is a card.
Cards always have a clear hierarchy: **headline → price → supporting info →
action.** Don't break the order.

## Plan card

<Canvas of={CardStories.PlanCards} />

Three variants of the same plan row:

- **Entry** — neutral badge, outline action.
- **Mid** — 2px brand-500 border, "Most popular" pill sitting on the edge, brand price, primary action.
- **Premium** — feature card. `variant="feature"` applies the signature gradient
  (`brand-500 → accent-500`). Reserve the gradient for feature cards, AI badges,
  and the logo mark — nowhere else.

## Device card

<Canvas of={CardStories.DeviceCards} />

Devices use a dedicated layout (not the base `Card`) — `grid-cols-[64px_1fr_auto]`,
thumbnail left, meta centre, price right. The price always uses
`font-display font-bold` in `brand-700`.
